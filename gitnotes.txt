--- Things learned in the git
1. Create a New Repository on GitHub
  a). Go to GitHub.
  b). Click the + icon(top-right) --> New Repository
  c). Fill In
      Repository Name :
      Description : "A sandbox to practice Git commands and workflows"
      Choose Public or Private.
      Check Initialize with a README.
  d). Click Create repository.

2). Clone the Repo Locally

  git clone https://github.com/YOUR_USERNAME/RepositoryName.git
Here RepositoryName is the name of the repository which you have created.

 cd RepositoryName

3. Start Practicing Git Commands
Create a file and commit it:
echo "# Git Practice Lab" > intro.md
git add intro.md
git commit -m "Add intro file"
git push origin main


Try branching:
git checkout -b feature-branch
echo "Testing branching" > branch.txt
git add branch.txt
git commit -m "Add branch test file"
git push origin feature-branch
